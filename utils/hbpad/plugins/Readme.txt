HbPad plugins
~~~~~~~~~~~~~

  HbPad plugins are compiled in hrb format programs, which may be
executed from HbPad, tools/pluguns menu.

  Plugins.ini file should contain info about plugins in use.

  plug_cipher - a plugin to encrypt/decrypt text snippets.
  To compile it to hrb from prg, use the following command:

      harbour plug_cipher.prg -n -gh
